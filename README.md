# xp-vpn-stack
Debian 13 L2TP/IPsec Full-Tunnel VPN for Windows XP/Vista with SQL/RADIUS AAA, nftables policy, per-PPP QoS, AdGuard+Unbound, internal OpenResty/PHP webpanel, MITM HTTPS blockpage (CA + MyPal NSS trust), NTP strategy, and onboarding gates (claim + email verify) with walled-garden enforcement.
